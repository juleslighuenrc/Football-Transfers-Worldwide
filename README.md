# Football-Transfers-Worldwide-Excel-Analysis

## Overview
Small Excel project analyzing football transfers worldwide.
The goal was to clean the data, handle missing values, create a Europe-focused table, and visualize top-spending countries.

This project is meant to showcase Excel data cleaning, statistics, and visualization skills.

## What was done
- Fixed missing and inconsistent values
- Created a separate table for European transfers
- Applied statistical methods
- Built a visualization showing top spending clubs

## File
- `football-transfers-excel-analysis.xlsx`
  - `Database` –  cleaned data
  - `Countries` - data used to fill missing values
  - `European Transfers` – filtered table for European transfers amount
  - `European Transfers by Country` - table creation and application of statistical operations
  - `Visualization Top 5 Spenders` – final chart

## Excel skills demonstrated
- Data cleaning and filtering
- Handling missing values
- Table creation
- Statistics
- Basic data transformation
- Charts and visualizations

## Notes
This is a small, self-contained project completed in a few hours as a skills demonstration.
